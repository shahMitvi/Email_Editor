# Email Editor (React Email Builder)

A drag-and-drop Email Template Builder built using React.
This project allows users to design email templates visually by adding and customizing components like text, images, buttons, and QR codes.

---

## Features

- Drag and Drop Email Builder
- Edit text, images, buttons, and QR code components
- Dynamic variable support for QR codes
- Undo / Redo functionality
- Responsive preview (Desktop, Tablet, Mobile)
- Export email template as HTML

---

## Tech Stack

Frontend:

- React
- Vite
- TypeScript
- Zustand (state management)
- React Moveable (resize, drag, rotate components)

Version Control:

- Git
- GitHub

## Installation

Clone the repository:

git clone https://github.com/shahMitvi/Email_Editor.git

Go to project folder:

cd Email_Editor/client

Install dependencies:

npm install

---

## Running the Project

Start the development server:

npm run dev

The application will run at:

http://localhost:5173

---

## Build for Production

To create a production build:

npm run build

Preview production build:

npm run preview
